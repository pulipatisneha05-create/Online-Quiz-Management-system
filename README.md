The Online Quiz Management System is a Java-based desktop application developed to conduct online quizzes in an interactive and user-friendly manner. The system allows users to enter their details, read quiz rules, answer multiple-choice questions, and view their final score instantly. The application is designed using Java Swing for GUI development and provides an efficient way to manage and evaluate quizzes automatically.
The project aims to simplify the quiz process by automating question handling, score calculation, and result generation while providing a smooth graphical user interface for users.
Features:
User registration with name input
Rules and instructions screen
Multiple-choice quiz questions
Automatic score calculation
Skip question functionality
Instant result generation
Graphical User Interface (GUI) using Java Swing
Interactive buttons and navigation system
Technologies Used:
Programming Language
Java
GUI Development
Java Swing
AWT (Abstract Window Toolkit)
Concepts Used
Event Handling
Object-Oriented Programming (OOP)
Arrays
JFrame & JPanel
ActionListener
Development Environment
VS Code / Eclipse

Project Workflow:
User enters name on welcome screen
Rules screen is displayed
Quiz starts with multiple-choice questions
User selects or skips answers
System evaluates answers automatically
Score is calculated dynamically
Final result screen is displayed
